vii: the css3 animation generator

Instructions

1) Type your commands:

vii. ('')

Load a pre-made animation: *

*Animations are taken from CSS3 Animation Cheat Sheet, Animate.css, Magic Animations, Hover.css and liffect.

2) Quick mode:

  • Open your browser console.
  • Type commands using vii.k('') for keyframes, vii.t('') for transitions, vii.kt('') for keyframed transitions and vii.it('') for interactive transitions.
  • Copy the generated code into a .css file.
