vii: The css3 Animation Generator

This tool helps you write the code for CSS animations really fast. No need to worry about prefixes or the correct transform strings or even your keyframe names!

Instructions

Type your commands:

  • Type commands using vii.k('') for keyframes, vii.t('') for transitions, vii.kt('') for keyframed transitions and vii.it('') for interactive transitions.
  • Press the GENERATE button to preview your animation.
  • Copy the generated code into a .css file.
